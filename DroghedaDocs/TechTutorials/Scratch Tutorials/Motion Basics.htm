Motion Basics

From Kata - CoderDojo
Jump to: navigation, search

Created by Pam O'Brien


These scripts demonstrate the basics of moving a character (or sprite) around the screen

Note: To find yellow blocks go to 'Control' and to find blue blocks go to 'Motion'



Make the character move 10 steps when the a key is pressed


Note : to find 'when a key pressed' go to control and choose 'when space key pressed' and click on the arrow beside space to find the a. You can change the number of steps to move by clicking on the 10 and changing




Turn the character 15 degrees in a clockwise direction when the b key is pressed


Note : you can change the number of degrees by clicking on the 15 and changing



Turn the character 15 degrees in an anti-clockwise direction when the c key is pressed

Note : you can change the number of degrees by clicking on the 15 and changing



Make the character go to an x and y co-ordinate when the d key is pressed

Note: you can change the x and y co-ordinates by clicking on the 150 and the -100 and changing



Make the character glide to an x and y co-ordinate when the e key is pressed

Note: you can change the x and y co-ordinates by clicking on the -150 and the 120 and changing. You can change how quickly the character glides by changing the number of seconds from 1



Make the character move left when the left arrow is pressed

Note: to get a character to move left change the x value by a negative number



Make the character move right when the right arrow is pressed

Note: to get a character to move right change the x value by a positive number



Make the character move up when the up arrow is pressed

Note: to get a character to move up change the y value by a positive number



Make the character move down when the down arrow is pressed

Note: to get a character to move down change the y value by a negative number



You can put all of these script snippets in one script for a sprite or character and get the sprite to do difference things depending on which key has been pressed
